# dotfiles-manjaro-awesome-18.1.3

Dotfiles management following this tutorial: https://www.atlassian.com/git/tutorials/dotfiles

# Setup

- sudo pacman-mirrors -f 5 && sudo pacman -Syyu
- sudo pacman -S yay
- sudo yay -S lsd
- sudo yay -S nvim
- sudo yay -S nodejs
- sudo yay -S npm
- sudo pacman -R compton
